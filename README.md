
# HubSpot AI Automation System

## 📌 Overview

A complete CRM transformation project where we optimized HubSpot and implemented AI-powered automation workflows.

---

## 🚧 Problem

- Messy CRM setup
- Poor lead tracking
- Manual follow-ups
- No automation
- Low visibility into pipeline

---

## ⚙️ Solution

### CRM Restructuring
- Cleaned and reorganized HubSpot
- Designed structured pipelines
- Created custom properties

### Automation Workflows
- Lead nurturing workflows
- Task automation
- Logic-based triggers

### AI Voice Agent
- NLP-based outbound calling
- Lead qualification automation

### Integration
- Trustpilot integration with CRM

---
